# OpenClaw State Backup

Versioned backup and restore skill for OpenClaw state, with checksum verification, dry-run restore plans, include/exclude filters, and automatic pre-restore rollback snapshots.

## Contents

- `SKILL.md`
- `scripts/backup_state.py`
- `scripts/restore_state.py`

## Notes

This repository contains only the standalone `openclaw-state-backup` skill, separated from the larger private workspace to avoid publishing unrelated or personal files.
